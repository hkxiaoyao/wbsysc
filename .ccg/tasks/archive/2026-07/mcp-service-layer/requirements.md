# 已确认需求

- 一个租户可创建多个第三方连接实例和多个 MCP 服务入口。
- 每个 MCP 服务入口显式选择多个连接实例中的多个工具。
- 同一入口内工具对外名称唯一，默认由连接别名和源工具名生成，并允许显式修改。
- 一个 OpenAPI MCP 工具可顺序编排 1 至 N 个接口，支持输入和前序步骤结果引用、结果合并、失败即停。
- 代码连接器由平台安装并启用；租户只能创建实例，不能上传或运行自定义代码。
- 租户采用单账号，以租户标识和密码登录，不建设租户成员或角色体系。
- MCP Token 绑定服务入口，一个入口可签发多个 Token。
- 平台管理员和所属租户登录后都可查看、复制完整的新服务 Token，不要求二次输入密码。
- 旧连接 Token 和旧 MCP 地址在兼容期继续可用。
- 后续外部模型只调用 CC（Claude Code），不调用 Gemini。
