# 企微接入配置清单（每个新客户/租户上线必走）

> 每接入一个客户，按本清单在客户企微后台完成配置，缺一不可。
> 联调记录：2026-07-12，错误码对照来自企微官方文档 + 微信开放社区。

## 一、凭证准备
- [ ] 获取客户企业 `corpid` + 自建应用 `secret`
- [ ] secret 加密存入 KMS（一期存 `.env` 但 gitignore）
- [ ] 记录自建应用 `agentid`

## 二、企业可信 IP 白名单（所有应用接口必需）
**路径**：管理后台 → 应用管理 → 对应自建应用 → "企业可信IP"
- [ ] 加入中转服务器公网 IP
- [ ] 加入联调开发机 IP
- 错误码对照：未加白 → `60020 not allow to access from your ip`

## 三、审批接口权限（调审批类工具必需）
**路径**：管理后台 → 应用管理 → 【审批】应用 → API
- [ ] 「可调用接口的应用」→ 勾选你的自建应用
- [ ] 「审批数据权限」→ 授权应用允许提交/读取审批单据
- [ ] 自建应用 → 可见范围 → **包含要查询的审批申请人**（否则仍报 301055）
- 错误码对照：
  - `301055 no approval auth` = 应用未配置进审批的可调用应用，或申请人不在自建应用可见范围
  - `301056` = 审批应用已停用

## 四、汇报接口权限（调汇报类工具必需）
**路径**：管理后台 → 应用管理 → 【汇报】应用 → API → "可调用接口的应用"
- [ ] 选择你的自建应用
- 模板 ID 获取：汇报应用 → 某个汇报内容设置页 → 点"汇报名称"获取 template_id

## 五、智能表格接口权限（调智能表格工具必需）
**路径**：自建应用 → 应用配置
- [ ] 开通「文档」「智能表格」应用权限
- [ ] 准备真实测试数据：企业微信里建一个智能表格 → 从 URL 取 `docid` 和 `sheet_id`
- 错误码对照：`48002 api forbidden` = 应用未开文档/智能表格权限
- ⚠️ **一期搁置**：企微docid仅API创建表可得，成员手工存量表无法读（见 plan）

## 五-2、打卡接口权限（调打卡工具必需）
**接口**：`/cgi-bin/checkin/getcheckindata`（获应用可见范围内员工打卡记录）
- [ ] 自建应用配置到「打卡 - 可调用接口的应用」（路径同审批/汇报，打卡应用→API）
- [ ] 应用可见范围**包含要查打卡的员工**（打卡只返回可见范围内成员数据）
- [ ] **useridlist 来源**（二选一）：
  - **A.自动拉（推荐）**：配通讯录同步secret → 自动调 `user/list_id` 拉全企业userid喂打卡
    - 路径：企微后台 → 管理工具 → 通讯录同步 → 开启API接口同步 → 拿通讯录同步secret
    - tenant_init 加 `--contact-secret XXXX`
    - 注意：list_id **必须用通讯录同步secret**，不能用自建应用secret（会报 40001）
  - **B.手动配**：`--checkin-userids "userA,userB"`（适合人少/精确控制）
- 限制：时间跨度≤30天、useridlist≤100(自动分批)、600次/分
- 返回：一次拿全 `checkindata`（含 checkin_type/time/location/exception_type 等，无需二次拉详情）
- 降级：自动拉失败(如secret失效)→自动回退手动配，不中断同步

**重要：打卡可见范围**（自动拉 userid 后常见坑）
- `user/list_id` 拉的是**全企业 userid**，但打卡接口只返回**在打卡应用/自建应用可见范围内**的员工数据
- 不在可见范围的员工调打卡会报 `301021 request userid error`
- 系统已做**逐人容错**：单人越权静默跳过，不影响其他人打卡拉取
- 若要查某人打卡但报 301021 → 把该员工加入自建应用/打卡应用的**可见范围**

## 六、错误码速查
| 错误码 | 含义 | 处理 |
|--------|------|------|
| 0 | 成功 | - |
| 60020 | IP 未加可信白名单 | 加 IP（见二） |
| 301055 | 无审批数据拉取权限 | 见三 |
| 301056 | 审批应用已停用 | 启用审批应用 |
| 48002 | 应用无对应 API 权限 | 见四/五 |
| 301112 | 查询时间范围过大 | 缩小到 ≤31天 |
